# External HUD Projector

This tool is the host program used to display sensor information in tandem when using the [External HUD client](https://github.com/rwqfsfasxc100/External-HUD-Client) mod for Delta-V: Rings of Saturn.


# Installation

1. Head to the latest release page at https://github.com/rwqfsfasxc100/External-HUD-Projector/releases/latest
2. Download the release for your operating system
3. Unpack zip file
4. Run the executable (Linux users will need to `chmod` permissions as usual for applications)
